# Redline v1.19.0 — New features (flow tools)

## Features

1. **Keyboard flow** — when not typing:  
   - `1`–`5` path · `C` complete step · `N` capture · `U` undo complete · `?` shortcuts sheet · `⌘K` Spark  
2. **Copy pack brief** — Pack page clipboard markdown for client handoff  
3. **Session resume** — one quiet banner per browser session with project + next step  
4. **Tools → Keyboard** menu entry  

## Non-goals
- Full command palette  
- Cloud share links
